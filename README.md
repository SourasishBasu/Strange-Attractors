# Strange-Attractors
Generating 3D plots of Strange Attractors using matplotlib
<br></br>

## Prerequisites:

- Python 3.7 & up
- Open Command prompt and type:

  ```bash
  pip install matplotlib --user
  ```

### To run in terminal:
- Open Powershell in the local repository folder
- Type:

  ```bash
   python main.py
  ```
